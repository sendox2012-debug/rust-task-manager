# 🦀 Rust  Task Manager

Красивый, быстрый и легкий менеджер задач для терминала с поддержкой настоящих RGB-цветов, ссылок и интуитивного управления. Написан на чистом Rust без тяжелых зависимостей.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Rust](https://img.shields.io/badge/rust-v1.70+-orange.svg)

## ✨ Особенности

- 🎨 **Настоящий RGB**: Градиентные цвета и неоновый стиль в терминале.
- ⚡ **Быстрый**: Мгновенный отклик, управление стрелками и горячими клавишами.
- 🔗 **Умные ссылки**: Открывайте URL задач прямо из терминала одним нажатием.
- 🛠 **Без зависимостей**: Работает на стандартной библиотеке Rust.
- 💾 **Кроссплатформенность**: Поддержка Windows, macOS и Linux.

## 🚀 Как скачать и запустить (Без компиляции!)

Вам **не нужно** устанавливать Rust или компиляторы. Просто скачайте готовый файл для вашей системы.

### 1. Перейдите в раздел Releases
Откройте страницу релизов этого репозитория:  
👉 **[Releases Page](../../releases)**

### 2. Скачайте файл для вашей ОС
Найдите последнюю версию (например, `v1.0.2`) и скачайте соответствующий файл:

| Операционная система | Файл для скачивания | Инструкция по запуску |
| :--- | :--- | :--- |
| **Windows** | `rust-task-manager-x86_64-pc-windows-msvc.exe` | 1. Скачайте файл.<br>2. Запустите двойным кликом.<br>*(Если Windows предупреждает: нажмите "Подробнее" → "Все равно запустить")* |
| **macOS** | `rust-task-manager-x86_64-apple-darwin` | 1. Скачайте файл.<br>2. Откройте терминал в папке загрузки.<br>3. Дайте разрешение: `chmod +x rust-task-manager-x86_64-apple-darwin`<br>4. Запустите: `./rust-task-manager-x86_64-apple-darwin` |
| **Linux** | `rust-task-manager-x86_64-unknown-linux-gnu` | 1. Скачайте файл.<br>2. Откройте терминал.<br>3. Дайте разрешение: `chmod +x rust-task-manager-x86_64-unknown-linux-gnu`<br>4. Запустите: `./rust-task-manager-x86_64-unknown-linux-gnu` |

## 🛠 Для разработчиков: Сборка из исходников

Если вы хотите изменить код или собрать программу самостоятельно:

### Требования
- Установленный [Rust](https://rustup.rs/) (версия 1.70+)

### Команды сборки
```bash
# Клонируйте репозиторий
git clone https://github.com/sendox2012-debug/rust-task-manager.git
cd rust-task-manager

# Запуск в режиме разработки
cargo run

# Сборка оптимизированной версии (релиз)
cargo build --release
